步骤5: 生成关联规则并计算提升度

从频繁项集{牛奶, 面包}生成关联规则:

提升度公式: Lift(A → B) = P(B|A)/P(B) = P(A∩B)/(P(A)×P(B))

规则置信度提升度是否强规则解释
牛奶 → 面包 4/6 ≈ 66.67% 0.4/(0.6×0.5) ≈ 1.33 是 (置信度≥50%) 提升度>1，正相关
面包 → 牛奶 4/5 = 80% 0.4/(0.5×0.6) ≈ 1.33 是 (置信度≥50%) 提升度>1，正相关

提升度解释：

  • 提升度 = 1：A和B独立
  • 提升度 > 1：A和B正相关，一起出现比随机出现更频繁
  • 提升度 < 1：A和B负相关，一起出现比随机出现更少

在我们的例子中，两条规则的提升度都是1.33，说明牛奶和面包之间存在正相关关系。
